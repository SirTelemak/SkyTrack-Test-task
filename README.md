# SkyTrack-Test-task
Wikipedia recursive scraper
## Usage

```
python main.py --url [url_to_parse] --depth [max_depth]
```
### Available commands

`url` - URL of wikipedia article to be parsed
`depth` - maximum recursion depth
`help` - view help
